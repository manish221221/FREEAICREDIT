# **App Name**: AI Hub

## Core Features:

- Key Management: Key Management: Allow users to add, edit, and prioritize API keys from various AI platforms (OpenAI, Anthropic, Google, etc.).
- Model Selection: Model Selection: Enable users to select from available models across different providers through a unified interface.
- Intelligent Routing: Intelligent Routing:  The app, acting as a tool, will automatically route requests to the optimal API key and model based on availability, priority, and routing policies.
- Status Monitoring: Status Monitoring: Provide a transparent UI to display which key/model handled a request, along with reasons for any switches (e.g., rate limits).
- Mini-Agent Execution: Mini-Agent Execution:  The app will run user-defined task flows (mini-agents) that can call models and leverage device capabilities (clipboard, notifications, share sheet).
- Agent Creation: Agent Creation UI: a gallery of agent templates to get people started, plus a bare-bones UI to define simple multi-step automations.
- No-Backend Mode: No-Backend Mode: Support a mode where all API calls are made directly from the device to vendors, using user-entered keys stored securely.

## Style Guidelines:

- Primary color: Dark indigo (#4B0082) to convey technical sophistication and intelligence. 
- Background color: Very light, desaturated lavender (#F0F8FF) for a clean and unobtrusive feel. This choice supports both light and dark mode preferences without harsh contrast. 
- Accent color: Bright yellow-gold (#FFD700) to highlight interactive elements and key information, providing clear visual cues and an element of understated luxury.
- Font pairing: 'Space Grotesk' (sans-serif) for headlines and 'Inter' (sans-serif) for body text. Space Grotesk will lend a techy feel, while Inter keeps things readable in longer passages.
- Use a set of minimalistic, vector-based icons that are consistent in style, and avoid the use of color on them, except on key interactive elements.
- Maintain a clean and well-spaced layout with clear visual hierarchy. Prioritize important information and features while minimizing clutter.
- Incorporate subtle transitions and animations to provide feedback and enhance the user experience. Focus on speed and smoothness to avoid feeling sluggish.
